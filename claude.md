# Journey Project Rules
- **Design System:** Always follow the guidelines in `design-system.md`.
- **Tech Stack:** React, Tailwind CSS, Vite, Phosphor Icons.
- **Styling:** Use Tailwind utility classes. No inline styles.
- Use only the hex codes defined in `design-system.md`. Never use Tailwind's default color palette (e.g., no text-gray-500, no bg-blue-600). If a color is needed that isn't in the spec, ask me first.
